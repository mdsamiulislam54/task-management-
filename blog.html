What is event bubbling in the DOM?

Event Bubbling is a concept in the DOM that describes how events propagate through the DOM tree. When an event is triggered on an element, the event doesn't just affect the element that triggered it but also "bubbles up" to its parent elements. This propagation continues all the way to the root of the DOM tree unless it is explicitly stopped.

What is event delegation in the DOM?

Event delegation is a pattern in the DOM that allows you to attach event listeners to multiple parent elements and then help those parent elements handle events on their child elements.
